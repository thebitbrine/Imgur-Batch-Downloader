# Imgur-Batch-Downloader
### Basic Auto Batch Image Downloader For Imgur Written In C#

#### Features:
* Auto Folders
* Duplicate Detection (Experimental)
* Albums Only Option
* Auto Mode
* Minimal
* Minimum Image Count (For Albums)
* Auto Crash
* Buggy
